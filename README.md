### 저장소 사용 방법
- 자동완성 사용하지 않고, 책/인터넷 참조하지 않고 순수 손으로 작성해보기
- 이해하여 손으로 아키텍쳐를 그리고, 논리 구성이 가능할 때까지 계속 반복
